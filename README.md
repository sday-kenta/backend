Запуск проекта:
git clone -b feature/rubrics https://github.com/sday-kenta/backend.git

В папке проекта:
docker compose -d --build
если билдить проект заново не нужно, флаг --build опустить

Дока доступна по адресу:
http://localhost:8080/swagger/index.html#/
